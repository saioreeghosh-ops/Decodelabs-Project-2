# Decodelabs-Project-2
# Sales Data Analysis Using Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a sales dataset using Python. The dataset was cleaned, analyzed, and visualized to extract meaningful business insights.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Features

- Data Cleaning and Preprocessing
- Missing Value Analysis
- Statistical Analysis
- Sales Analysis by Product
- Correlation Analysis
- Outlier Detection using IQR
- Data Visualization

## Analysis Performed

### Data Cleaning
- Removed duplicate records
- Handled missing values
- Verified data consistency

### Statistical Analysis
- Calculated mean and median values
- Analyzed product-wise sales performance

### Outlier Detection
- Identified unusual sales values using the Interquartile Range (IQR) method

### Visualizations
- Bar Chart
- Histogram
- Pie Chart
- Box Plot

## Learning Outcomes

- Data Cleaning with Pandas
- Exploratory Data Analysis (EDA)
- Data Visualization with Matplotlib
- Statistical Data Analysis
- Business Insight Generation
